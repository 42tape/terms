# 42Tape

42Tape은 차세대 초저비용·고보안 분산 클라우드 스토리지를 개발하는 기술 기업입니다.
웹2 수준의 사용성을 유지하면서, 백엔드에서는 DePIN 기반 저장소(Filecoin PDP, Sia, Storj 등), 
Proxy Re-Encryption, MPC(Multi-party computation), AI 기반 데이터 자동 분류 기술 등이 결합된 
‘Lifetime Cloud Storage’ 서비스를 구축하고 있습니다.
우리는 기존 클라우드의 한계를 해결하고, 
사용자에게 안전하고 빠르며 저렴한 장기 저장 공간을 제공하는 것을 목표로 합니다.
